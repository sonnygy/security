# Изучение алгоритмов RSA 

## Описание используемого метода
**RSA** – ассиметричный метод шифрования, использующий два ключа:
* Открытый ключ (e, n) — для шифрования 
* Закрытый ключ (d, n) — для расшифрования
**Основная идея:**
* сообщение превращается в число 
* это число возводится в степень по модулю 
* безопасность основана на сложности разложения числа на множители 

### Алгоритм работы программы
**Генерация ключа:**
1.	Выбор двух простых чисел p и q, вычисление n = p * q
2.	Подсчет функции Эйлера φ(n) = (p - 1)(q - 1) и выбор числа e
3.	Нахождение d, такого что (e * d) mod φ(n) = 1
**Формирование ключей:**
* Открытый ключ (e, n)
* Закрытый ключ (d, n)
**Шифрование:**
Представляем сообщение как число M
C = M^e mod n
**Дешифрование:**
M = C^d mod n
