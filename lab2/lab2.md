# Шифрование данных при помощи генератора псевдослучайных чисел

## Описание используемого метода
### Гаммирование с обратной связью
**Метод гаммирования** — это шифрование, при котором:
* генерируется псевдослучайная последовательность (гамма) 
* она накладывается на исходный текст с помощью операции XOR (mod 2) 
* используется обратная связь, то есть:
    * следующий элемент гаммы зависит от контрольной суммы предыдущего блока
**Генерация гаммы с помощью ПСЧ:**
Используется формула линейного конгруэнтного генератора:
T(i + 1) = (A ⋅ T(i) + C) mod M
где:
* A, C — константы 
* T(0) — начальное значение (seed) 
* M = 2^b
* b = 8
**Шифрование**
C = M ⊕ G
**Дешифрование**
M = C ⊕ G
где: M — исходный текст, G — гамма, C — шифртекст

### Алгоритм работы программы
**Шифрование:**
* Перевести текст в бинарный вид 
* Сгенерировать первый элемент гаммы:
    * T(1) = (A ⋅ T(0) + C) mod M
* Найти контрольную сумму (кол-во единиц) 
* Сгенерировать следующий элемент:
    * T(i + 1) = (A ⋅ S + C) mod M
* Выполнить XOR 
* Повторить
