# Сети фейстеля

## Описание используемого метода
**Сеть Фейстеля** — это структура, используемая при построении блочных шифров. Она преобразует входной блок данных фиксированной длины в зашифрованный блок той же длины.
* данные делятся на части (ветви); 
* выполняется несколько раундов преобразований; 
* используется функция F; 
* применяется операция XOR (сложение по модулю 2); 

### Описание исходных данных
**Количество ветвей:** 2 (классическая схема)
**Количество раундов:** 18
**Образующая функция:** арифметический сдвиг влево

### Алгоритм работы программы
* Разделить входной текст на два блока (L и R). 
* Для каждого раунда: 
    * вычислить F(R, ключ); 
    * выполнить L ⊕ F(R); 
    * поменять местами L и R. 
* После завершения раундов объединить блоки. 
* Для дешифрования выполнить те же действия в обратном порядке.
