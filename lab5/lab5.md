# Создание электронной подписи

## Описание используемого метода
#### Подготовка ключей
* p — большое простое число 
* q — простой делитель (p-1) 
* a — число такое, что a ^ q mod p = 1
* x — секретный ключ (случайное число от 1 до q) 
* y = a ^ x mod p — открытый ключ

#### Создание подписи
Для сообщения m:
* Считаем хэш: 
    * h = hash(m), если h = 0, делаем h = 1 
* Генерируем случайное число k (1 < k < q) 
* Считаем: 
    * r = (a^k mod p) mod q, если r = 0 → выбрать новый k 
* Считаем подпись: 
    * s = (x*r + k*h) mod q , если s = 0 → выбрать новый k 
Подпись = (r, s)

#### Проверка подписи
* Проверяем: 
    * 0 < r < q и 0 < s < q 
* Считаем: 
    * h = H(m) 
    * v = h ^ (q - 2) mod q 
    * z1 = (s * v) mod q 
    * z2 = ((q - r) * v) mod q 
    * u = ((a^z1 * y^z2) mod p) mod q 
* Если u == r - подпись верна
Подпись зависит:
* от сообщения 
* от секретного ключа 
Проверка:
* использует только открытый ключ 
* подтверждает, что: 
    * сообщение не изменилось 
    * его подписал именно владелец ключа
